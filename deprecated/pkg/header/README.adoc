== Dfx header variables

[cols="1,1,1"]
|===
| Variable
| Group code
| Description

| $ACADMAINTVER
| 70
| maintenance (should be ignored)

| $ACADVER
| 1
| Autocad drawing database version number (shown below)

| $ANGBASE
| 50
| Specifies the compass direction east as the 0 angle.

| $ANGDIR
| 70
| 1 -> Clockwise and 0 -> Counterclockwise angles


|===
